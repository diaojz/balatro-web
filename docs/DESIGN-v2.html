给开发的实现要点

  • ① Vue 3 组件一律用 <script setup lang="ts">，推荐命名：SalesCard.vue / SalesShop.vue / Hud.vue / HandCard.vue / GameBoard.vue
  • ② 计分管道用 pure function：calcScore(hand, jokers, ctx) → ScoreResult，方便 Vitest 单元测试，不要把逻辑散在组件 computed 里
  • ③ 销售牌效果用函数式：effect: (ctx: ScoreContext) => ScoreContext，所有 20 张牌统一挂此接口，主循环顺序调用即可
  • ④ 大屏布局：body { display:grid; place-items:center }，.game-viewport 用 min(100vw, 1080px)，超宽屏不变形
  • ⑤ HUD 避免闪烁：直接绑定 reactive state（gameState.score 等），不要用 v-html 或 innerHTML
  • ⑥ 销售牌槽位用 <TransitionGroup name="slot"> 做装备/出售动画，配合 translate + opacity 过渡
  • ⑦ 销售牌触发高亮：计分管道返回 triggeredJokerIds: string[]，组件根据此列表短暂加 .glow class（200ms 脉冲）
  • ⑧ GitHub Pages 部署：Vite 配置 base: '/balatro-web/'，Actions 跑 build 后推 gh-pages 分支
