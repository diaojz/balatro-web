1项目目标

在 V1 计分引擎基础上，接入"选盲注 → 打牌 → 商店 → 购买 Joker → 推进 Ante"完整循环，让玩家能通过收集 Joker 组合指数级拉高分数，体验 Balatro 核心乐趣。

技术约束（继承自 V1）

  • 单文件 HTML + CSS + JS，无构建工具，无框架，无外部依赖（除 Google Fonts）
  • 游戏视口固定 1080×640，不做移动端适配
  • V1 Card 结构 { suit, rank, value } 向下兼容，V2 追加可选字段不破坏现有逻辑

Ante 分数常量

const ANTE_BASE = { 1: 300, 2: 800, 3: 2000 };
const BLIND_MULT = { small: 1, big: 1.5, boss: 2 };
// Ante1 Small=300, Big=450, Boss=600
// Ante2 Small=800, Big=1200, Boss=1600
// Ante3 Small=2000, Big=3000, Boss=4000
function getTargetScore(ante, blind) {
  return Math.floor(ANTE_BASE[ante] * BLIND_MULT[blind]);
}

计分公式

最终得分 = (hand_chips + Σ scoring_card_chips + Σ joker_addChips) × (hand_mult + Σ joker_addMult) × Π joker_xMult
function calcScore(selectedCards) {
  const handType = evaluateHand(selectedCards);
  const level = state.handLevels[handType.key];
  const scoringCards = getScoringCards(selectedCards, handType);

  let chips = handType.baseChips(level);
  for (const c of scoringCards) {
    if (!c.debuffed) chips += c.value;
  }
  let mult = handType.baseMult(level);
  let xMultProduct = 1;

  for (const joker of state.jokers) {
    const { addChips = 0, addMult = 0, xMult = 1 } = joker.effect(ctx);
    chips += addChips;
    mult  += addMult;
    xMultProduct *= xMult;  // ×Mult 最后统一做乘法
  }

  return Math.floor(chips * mult * xMultProduct);
}

4核心流程

  1. 游戏开始：ante=1, money=0, jokers=[], 洗牌，phase='blind_select'
  2. 盲注选择（phase=blind_select）：显示 Small/Big/Boss 目标分和奖励；Small/Big 可跳过（不给奖励）；Boss 不可跳过
  3. 打牌阶段（phase=playing）：发 8 张手牌，设 targetScore；玩家出牌/弃牌；达到 targetScore → 本 Blind 胜利，进结算
  4. 结算：计算基础奖励 + 剩余出牌弃牌奖励 + 利息，money 更新，显示明细
  5. 商店（phase=shop）：2 格 Joker + 1 格 Coming Soon 占位 + 刷新 + 继续
  6. 3 个 Blind 均通过 → ante++，回到步骤 2；ante 推进到 4 → phase='win'
  7. 任意 Blind 出牌耗尽且未达分 → phase='lose'，显示进度

5决策记录（2026-05-16 已拍板）

  • D1 · Joker 数量 → 固定 8 张
    商店每次随机展示 2 张，底池 8 张可控，够跑通主循环。V2.1+ 再扩库。
  • D2 · 跳过 Blind 的奖励 → 不给任何奖励
    跳过 = 放弃本轮金钱与利息，换取节省时间，是纯权衡而非纯收益。Tag 系统顺延 V2.1+。
  • D3 · 商店第 3 格 → Coming Soon 占位格
    展示随机塔罗或星球牌外观，标注 "Coming Soon"，不可交互，纯展示组件，不接计分逻辑。
